WeChat
===
您还在为自己的项目社交模块烦恼吗？ 您还在琢磨微信的功能是怎么实现的吗？ 别愁了，快来这看看吧！  
本项目为高仿微信，并在微信基础上对一些体验和UI上做了优化，实现功能有：  
好友之间文字聊天，表情，视频通话，语音，语音电话，发送文件等。
群聊，从通讯录添加好友，二维码扫描添加好友，好友之间二维码扫描转账，
微信公众账号消息推送、朋友圈等   
如果您对本项目感兴趣，欢迎加入我们开源大家庭！  
Family ！！！
